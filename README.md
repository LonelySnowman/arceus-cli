# arceus-cli

## 📕 简介

-   ✅ 快速生成 Vue3 框架模板 | [Vue3快速开发模板](https://github.com/LonelySnowman/sv3-template.git)
-   ✅ 快速生成 Vue3 框架模板（精简版） | [Vue3快速开发模板（精简版）](https://github.com/LonelySnowman/sv3-template/tree/thin)
-   ⏱ 更多模板 | (待更新)

## 📦 安装

```bash
  npm i arceus-cli -g
```

## 📕 使用

```bash
# 使用模板创建一个新项目
arceus create

# 执行项目文件夹名称并选择模板创建新项目
arceus create [prjName]

# 更新脚手架到最新版本
arceus update

# 查看脚手架版本
arceus -v
arceus --version
```
